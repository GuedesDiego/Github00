Entendido o conceito de Branch e Merge



Para criar uma Branch é muito simples, basta usar o comando git checkout -b e o nome da Branch


para voltar para Branch principal a master, bastar usar o comando git checkout e o nome master.

O conceito de Mergiar ou merge você usa o comando git merge e o nome da Branch que você ira espelhar, ou adicionar em sua Branch master
